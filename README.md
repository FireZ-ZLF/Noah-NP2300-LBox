Noah-NP2300-LBox

一个诺亚舟NP2300学习机拓展工具包

安装方法:
将安装文件复制到储存卡中,
执行lbox.sh 选择"安装(升级)",
然后再打开lbox.sh,选择"脚本"-"安装",
重启即可.
注意保留log文件夹 其余文件可删除

卸载方法:
将log文件夹与lbox.sh复制到同一目录下中,
执行lbox.sh 选择"卸载",
重启即可.

功能简介:
[命令执行器]
执行本地磁盘/工具/记事本/c.txt
可以选择QT对话框(默认)或电子书查看执行结果
保存的执行结果在储存卡/命令.txt
